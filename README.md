# OTP Login System
A simple OTP (One-Time Password) Login Authentication project built using Firebase Authentication.
This project is created for learning and demonstration purposes to show how OTP-based login systems work.

## Features

✔ Phone Number Login

✔ OTP Verification

✔ Firebase Authentication

✔ Loading State

✔ Input Validation

✔ Responsive Design

✔ Form Reset After Login

✔ CAPTCHA Integration

✔ OTP Visibility Toggle

## How It Works

1. Enter mobile number
2. Request OTP
3. Enter OTP
4. Verify
5. Login successful

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Authentication:
- Firebase Authentication

## Project Purpose

This project was created for educational purposes to learn:

- Authentication flow
- Firebase integration
- OTP verification
- Frontend development


## Notes

This project does NOT use real mobile number OTP delivery.

Instead, Firebase test phone numbers are used for demonstration and educational purposes.
You can manually configure a testing phone number and OTP inside Firebase Console.
